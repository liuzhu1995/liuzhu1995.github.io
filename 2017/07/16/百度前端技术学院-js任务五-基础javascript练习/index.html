如图，模拟一个队列，队列的每个元素是一个数字，初始队列为空

点击”右侧入”，将input中输入的数字从右侧插入队列中；
点击”左侧出”，读取并删除队列左侧第一个元素，并弹窗显示元素中数值；
点击”右侧出”，读取并删除队列又侧第一个元素，并弹窗显示元素中数值；
点击队列中任何一个元素，则该元素会被从队列中删除

队列元素数量最多限制为60个，当超过60个时，添加元素时alert出提示
