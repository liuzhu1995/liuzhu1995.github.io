jQuery中attr()与prop()

发表于 2017-07-04

attr()和prop()都是用来获取被选元素的方法，定义和用法都是一样的：
设置或返回被选元素的属性和值。
用attr()方法返回的属性是字符串，而用prop()返回的是true/false,当直接添加该属性名称就能生效的属性，可以直接使用prop()设置。

学习中遇到的两列问题 如下：

问题描述当选中其中一个后点击全选正确选中，但当点击全不选后，使用全选选中的可以取消，第一次单点选中的无法取消。

问题描述单击一行时，单选按钮被选中，但是当点击一遍后，单选按钮全部添加了checked=”checked”属性，当再次点击其它按钮时，单选按钮的选中状态会保持在最后一次点击的按钮

以上两个问题都是我在根据锋利的jQuery学习练习过程中出现的问题，并且之前都没有解决，今天在回过头研究attr()和prop()方法时，发现只要用prop()方法将attr()替换，问题不会出现问题。
所以当今后使用jQuery,应该用prop()方法获取和设置checked属性，disabled,selected也一样。
