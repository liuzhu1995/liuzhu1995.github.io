**节点信息**
childNodes可以获取任意节点的所有子节点集合
通过节点类型nodeType区分的有(element元素节点(1,节点类型)，attr属性节点(2),text文本节点(3),
comments注释(8),document(9))
element是node多种类型的其中一种，当nodeType == 1 时就是元素节点，它拥有id,class,children等属性。

subString(a,b)
用于提取字符串中介于两个指定下标之间的字符,它有两个参数,都是非负的证书,第一个要提取的字符串
的第一个字符,第二个可选,如果省略返回的字符串一直到字符串的结尾。

**innerHTML和innerText**
innerHTML 设置或返回指定元素开始标签与结束标签中的所有元素(自己和所有子元素)内容，包括标签元素。
innerText 设置或者返回指定元素开始标签与结束标签中的所有(自己和所有子元素)文本内容。

**push()**
向数组末尾添加一个或者多个元素，并且返回新长度。
最常用就是创建一个新数组var arr = []，在遍历的时候，将符合if条件的元素或者任意属性
添加到arr里面(arr.push());

**insetBefore()**
将一个新元素插入到一个现有元素的前面。
parentElement.insetBefore(new.Element,targetElement)
parentElement就是targetElement.parent

**removeChild()**
fatherObj.removeChild(childrenObj)
fatherObj要删除元素的元素对象，childrenObj要被删除的子元素对象

**childNodes和children**

childNodes返回节点的子元素集合，包括HTML节点，所有属性，文本节点。
可以通过nodeType来判断是哪种类型的节点，只有当nodeType==1时才是元素节点，2是属性节点，3是文本节点。
children它返回指定元素的子元素集合。但它只返回HTML节点，甚至不返回文本节点

split()和join()
split()用于把一个字符串分割成字符串数组 有两个参数，第一个必须 字符串或正则表达式 第二个可选返回数组的最大长度
join()将一个数组合并成字符串

replace()
用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式相匹配的子串。
stringObject.replace(regexp/substr,replacement)
regexp/substr 必须，规定字符串或需要替换的模式的regexp
replacement 必须，一个字符串值，替换文本或者生成替换文本的函数

eval()
可以用这个生成正则 例如 eval(“/“+ x +”/g”) x任意字符 用来匹配所有符合x值的字符串

foeEach()
遍历循环 for的简化和for()循环的效果一样
例如：var a = [,9,2,3,7,5,6];
a.forEach(function(e){
alert(e);
}) 会依次弹出数组里面的值

**阻止事件冒泡**
    e.stopPropagation()

**classList属性**
 返回元素的类名，可以用来增加或者删除指定的类名，它会自行判断这个类是否存在
 添加:element.classList.add();
 删除:element.classList.remove()
