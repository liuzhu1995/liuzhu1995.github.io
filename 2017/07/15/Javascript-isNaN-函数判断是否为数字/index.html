Javascript isNaN()函数判断是否为数字

发表于 2017-07-15

定义和用法
isNaN() 函数用于检查其参数是否是非数字值。
isNan()函数返回的是布尔值，当填入的参数为数值时返回的是false，非数值时为true。所以当要判断参数是否为数值时可以用
if(isNaN(参数){当返回为true时要执行的代码}
注意点：当isNaN()不输入任何参数的情况下都会返回true,所以在判断是否为数值之前都要先判断当要传入的参数是否为空值。
