关于用Hexo和Github建个人博客遇到的问题

发表于 2017-06-16

用了一天的时间,根据知乎大神吴润的的文章《GitHub+Hexo 搭建个人网站详细教程》自己搭建了一个人博客，并且如何将自己的域名替换Github的网址，这个教程里面也是有详细的介绍的，在自己搭建的过程中遇到了好多问题，有的耗费了好长时间，后面介绍的时候会跟着解决问题的网址的。
在把hexo和github绑定的时候，最容易出错的环节就是，给自己存放hexo代码的仓库起名。名字一定是要自己github名字后面加上github.io。例如：我的就是liuzhu1995.github.io。当时我自己没仔细看起名就是liuzhu1995后面的后缀是自己在hexo主题里面添加进去的，然后就无法和Github绑定。
折腾最久的一个问题就是关于Hexo 更换next主题之后,用localhost:4000进行本地预览，样式都没有错，但是当发布之后网页大片空白，当用控制台查看的时候发现js文件加载不了。当时快疯了都没找到原因，最后没管了，第二天打开网站正常。并且这个问题在我发布文章之后启用hexo g hexo d，网页出现了本地预览正常，打开网站没有文章，网站样式又无法加载，其实原因就是当把资源向github上传之后，会有一个X分钟的缓存时间，只要用hexo s本地预览没有问题，等过一段时间就好了。
