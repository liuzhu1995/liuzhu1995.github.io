仿豆瓣音乐播放器(原生js实现)

发表于 2017-08-07

任务描述
任务描述
模仿豆瓣音乐播放器，编写基于 html5的音乐播放器
顺序播放音乐列表支持循环，播放，暂停，快进，声音大小控制
有封面图片展示以及歌曲名称
音乐文件可以是本地或者来自网络
界面尽量做到简洁整齐不必和豆瓣界面一样

用了两天多的时间做完了这个任务，用了css3的@keyframes属性实现了图片的无限旋转，和audio的一些属性
实现的功能有：
1:头部显示正在播放的歌曲是列表总数的第几首
2:当网页加载完毕播放列表中第一首歌曲并且高亮显示列表对应行,右侧显示对应歌曲图片
歌曲名
3:当点击列表中任意一首歌曲时，高亮显示列表对应行，右侧显示对应歌曲图片歌曲名
4:改变音量大小
5:分别显示当前播放音乐的总时长和剩余时长
6:进度条 和 点击进度条任意位置跳转到音乐的对应位置进行播放(不支持Chrome浏览器)
7:播放,暂停(显示对应的图标),上一首,下一首功能
8:循环播放功能
9:点击对应歌曲的删除按钮时，删除列表中当前的歌曲
10:点击清空全部时,停止播放修改audio的链接,删除列表中所有的音乐
11:当前歌曲播放完毕自动播放下一首歌曲
