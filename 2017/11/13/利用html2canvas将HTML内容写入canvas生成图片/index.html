利用html2canvas将HTML内容写入canvas生成图片

发表于 2017-11-13

html2canvas工作原理

Html2canvas是一款前端插件,它的原理是将Dom节点在Canvas里边画出来。

限制

不支持iframe
不支持跨域图片
不能在浏览器插件中使用
部分浏览器上不支持SVG图片
不支持Flash

使用方法

首先下载然后引入html2canvas.min.js 下载地址

function takeScreenshot() { html2canvas(document.getElementById(element), { onrendered: function (canvas) { var url = canvas.toDataURL();//图片地址 document.getElementById('qq').appendChild(canvas); newCanva }, width: 300, height: 300 }); }
html2canvas 的使用非常简单，简单到只需要传入一个 DOM 元素到element位置，然后通过回调就可以拿到canvas

注意点

1.html2canvas 通过解析元素实际的样式来生成 canvas 图片内容，因此它对元素实际的布局和视觉显示有要求。如果要完整截图，最好将元素从文档流中独立出来（例如 position:absolute）

2.默认生成的 canvas 图片在 retina 设备上显示很模糊，处理成 2 倍图能解决这个问题。

3.html2canvas只能截取到浏览器中的可见区域。
解决方法在页面加载完成后，将准备截图的dom节点复制到一个隐藏在页面最下层的dom节点内，将该节点的层级设置为最底层，即：z-index最小，截图时，选取dom节点为该节点，就可以完整的将所有区域都截取到。
